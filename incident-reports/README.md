# Incident Reports

## Overview

This directory contains incident reports generated from simulated attacks performed within the Enterprise SOC Home Lab.

Each report follows a structured SOC investigation workflow, documenting the detection, analysis, impact assessment, and lessons learned from each simulated security event.

---

## Available Reports

| Incident | Description | Severity |
|----------|-------------|----------|
| IR-001 | Encoded PowerShell Execution | Medium |
| IR-002 | Registry Run Key Persistence | High |
| IR-003 | Suspicious Process Execution | Medium |

---

## Investigation Workflow

Each incident report includes:

- Executive Summary
- Incident Details
- Timeline
- Indicators of Compromise (IOCs)
- Investigation
- Impact Assessment
- MITRE ATT&CK Mapping
- Lessons Learned
- Conclusion
